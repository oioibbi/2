# 木鱼打罪（网页版打砖块）

纯前端（无构建）网页版小游戏：把小球替换成“圆形木鱼”，把砖块替换成“罪孽值”。打碎砖块会把等量罪孽值转化为“功德值”，但**只有通关小关（关卡/波次）才能把本关累计功德入账**；失败会失去本关未入账功德并重试本关。

## 运行

- 直接用浏览器打开 `index.html`
- 或用任意静态服务器（避免部分浏览器对本地存储的限制）

## 操作

- 鼠标/触屏移动：控制“法器”（挡板）
- 点击/轻触：发射木鱼
- `P`：暂停/继续
- `R`：重开本关

## 数据

- 功德榜、历史记录、累计功德：存于浏览器 `localStorage`（同一浏览器/同一设备）
